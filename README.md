# My First Flask App

A simple Flask web application built using Python and Pydroid 3.

## Features
- Home page
- About page
- Flask routing

## Technologies Used
- Python
- Flask
- HTML
- CSS

## How to Run
1. Install Flask
2. Run:
   python app.py
3. Open http://127.0.0.1:5000# my-first-flask-app
